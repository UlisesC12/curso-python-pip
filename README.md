# HOW TO SET UP

## Game Project

Para correr el juego debes de seguir las siguientes instrucciones en la terminal:

```sh
cd game
python3 main.py
```

O tambien correr directamente:

```sh
python3 game/main.py
```

## App Project

```sh
git clone
cd app
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
